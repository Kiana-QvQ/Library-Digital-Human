# Coze 语音克隆错误 4032 说明

## 错误信息

```
错误码: 4032
错误信息: Insufficient quota for the voice cloning resource pack.
中文含义: 语音克隆资源包配额不足
```

## 错误分析

### 错误类型
**配额/额度不足**，不是权限问题

### 原因说明
1. **配额已用完**：账户的语音克隆资源包配额已经全部使用完毕
2. **未购买资源包**：账户没有购买语音克隆资源包
3. **额度不足**：当前账户没有足够的语音克隆额度

### 验证方法
从你的测试结果可以看出：
- ✅ **API Key 有效**：音色列表查询成功（状态码 200）
- ✅ **权限正常**：可以正常调用 Coze API
- ❌ **配额不足**：语音克隆功能需要额外的资源包配额

## 解决方案

### 方案1：购买/续费资源包（推荐）
1. 前往 [Coze 控制台](https://www.coze.cn)
2. 进入 **配额管理** 或 **资源包管理**
3. 查看语音克隆资源包的使用情况
4. 购买或续费语音克隆资源包

### 方案2：联系客服申请配额
1. 联系 Coze 客服
2. 说明需要增加语音克隆配额
3. 提供账户信息和 logid：`20251223111040D60E3B6BC1FEBA09B2BB`

### 方案3：使用 Doubao（火山引擎）替代（已实现）
如果 Coze 配额问题无法解决，可以使用 Doubao 的语音克隆功能：

**优势：**
- ✅ 已完整实现（参考 `TTS_HS_API.py`）
- ✅ 支持语音克隆和状态查询
- ✅ 支持 WebSocket 流式 TTS
- ✅ 配额相对充足

**使用方法：**
