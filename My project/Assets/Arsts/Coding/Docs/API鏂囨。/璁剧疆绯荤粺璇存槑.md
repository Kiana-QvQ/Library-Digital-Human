# 设置系统 (Settings System)

## 📋 概述

这是一个完整的Unity设置管理系统，支持游戏、音频、UI、网络、语音、AI等多种设置类型，提供数据持久化、事件通知、UI集成等功能。

## 🏗️ 文件结构

```
Setting/
├── SettingsData.cs              # 设置数据结构定义
├── SettingsManager.cs           # 设置管理器（单例）
├── SettingsExample.cs           # 使用示例
├── UI/
│   ├── SettingsPanel.cs         # 主设置面板
│   ├── AudioSettingsPanel.cs    # 音频设置面板
│   └── VoiceSettingsPanel.cs    # 语音设置面板
├── Config/
│   └── default_settings.json    # 默认设置配置
└── README.md                    # 说明文档
```

## 🚀 快速开始

### 1. 初始化设置管理器

```csharp
// 在场景中创建一个GameObject，添加SettingsManager组件
// 或者通过代码获取实例
if (SettingsManager.Instance != null)
{
    var settings = SettingsManager.Instance.GetSettings();
}
```

### 2. 使用设置数据

```csharp
// 获取特定类型的设置
var voiceSettings = SettingsManager.Instance.GetVoiceSettings();
var audioSettings = SettingsManager.Instance.GetAudioSettings();

// 更新设置
var newVoiceSettings = new VoiceSettings();
newVoiceSettings.wakeWord = "你好小美";
SettingsManager.Instance.UpdateVoiceSettings(newVoiceSettings);
```

### 3. 监听设置变更

```csharp
// 监听所有设置变更
SettingsManager.OnSettingsChanged += (settings) => {
    Debug.Log("设置已更新");
};

// 监听特定类型设置变更
SettingsManager.OnVoiceSettingsChanged += (voiceSettings) => {
    Debug.Log($"唤醒词已更新: {voiceSettings.wakeWord}");
};
```

## 🔧 配置管理

### 默认设置
默认设置存储在 `Config/default_settings.json` 中，包含所有设置类型的默认值。

### 用户设置
用户设置自动保存到 `Application.persistentDataPath/user_settings.json`。

### 设置导入/导出
```csharp
// 导出设置
SettingsManager.Instance.ExportSettings("backup_settings.json");

// 导入设置
SettingsManager.Instance.ImportSettings("backup_settings.json");
```

## 📝 使用示例

### 基本用法
```csharp
public class MyScript : MonoBehaviour
{
    private void Start()
    {
        // 获取设置
        var voiceSettings = SettingsManager.Instance.GetVoiceSettings();
        
        // 应用设置到组件
        ApplyVoiceSettings(voiceSettings);
        
        // 监听设置变更
        SettingsManager.OnVoiceSettingsChanged += OnVoiceSettingsChanged;
    }
    
    private void ApplyVoiceSettings(VoiceSettings settings)
    {
        // 应用唤醒词
        var voiceManager = FindObjectOfType<VoiceControlScript>();
        if (voiceManager != null)
        {
            voiceManager.keywords = new string[] { settings.wakeWord };
        }
    }
    
    private void OnVoiceSettingsChanged(VoiceSettings newSettings)
    {
        ApplyVoiceSettings(newSettings);
    }
}
```

### 高级用法
```csharp
// 批量更新设置
var newSettings = new SettingsData();
newSettings.voice.wakeWord = "新唤醒词";
newSettings.audio.masterVolume = 0.8f;
SettingsManager.Instance.UpdateSettings(newSettings);

// 重置为默认设置
SettingsManager.Instance.ResetToDefault();

// 检查设置是否已加载
if (SettingsManager.Instance != null)
{
    var settings = SettingsManager.Instance.GetSettings();
    Debug.Log($"当前语言: {settings.game.language}");
}
```

## 🔄 事件系统

设置系统提供完整的事件通知机制：

- `OnSettingsChanged` - 任何设置变更时触发
- `OnGameSettingsChanged` - 游戏设置变更时触发
- `OnAudioSettingsChanged` - 音频设置变更时触发
- `OnUISettingsChanged` - UI设置变更时触发
- `OnNetworkSettingsChanged` - 网络设置变更时触发
- `OnVoiceSettingsChanged` - 语音设置变更时触发
- `OnAISettingsChanged` - AI设置变更时触发

## 🛠️ 扩展设置

### 添加新的设置类型
1. 在 `SettingsData.cs` 中定义新的设置类
2. 在 `SettingsData` 主类中添加新字段
3. 在 `SettingsManager.cs` 中添加对应的更新方法
4. 创建对应的UI面板（可选）

### 自定义设置验证
```csharp
private bool ValidateSettings(SettingsData settings)
{
    // 添加自定义验证逻辑
    if (settings.voice.wakeWord.Length < 2)
    {
        Debug.LogError("唤醒词长度不能少于2个字符");
        return false;
    }
    return true;
}
```

## 🐛 故障排除

### 常见问题
1. **设置未保存**: 检查 `Application.persistentDataPath` 权限
2. **JSON解析错误**: 确保JSON格式正确
3. **事件未触发**: 检查事件监听器是否正确注册
